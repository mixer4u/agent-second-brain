# Руководство по вводу данных тех. паспорта анкера

## Быстрый ввод через бота

Отправь в чат с ботом команду /do и данные анкера в свободной форме. Бот распарсит и сохранит в базу.

### Пример минимального ввода:
```
/do Добавь анкер в базу:
Hilti HIT-HY 200-A, М12, химический
Сверление: 14мм, глубина 110мм
Нагрузка: растяжение 18кН, срез 13кН
ETA-11/0493
```

### Пример полного ввода:
```
/do Добавь анкер в базу:
Производитель: Hilti
Продукт: HIT-HY 200-A
Тип: химический (гибридный)
Размер: M12
Материал: оцинкованная сталь
Смола: гибридная
ETA: ETA-11/0493

Монтаж:
- Диаметр сверления: 14 мм
- Глубина сверления: 110 мм
- Глубина заделки: 110 мм
- Мин. краевое: 60 мм
- Мин. шаг: 70 мм
- Момент затяжки: 60 Нм

Нагрузки (C20/25, без трещин):
- Растяжение: 18.0 кН
- Срез: 13.0 кН

Основание: бетон с трещинами, без трещин
Класс бетона: C20/25 — C50/60
Температура: -40°C ... +80°C
Время отверждения: 30 мин при 20°C
Сейсмика: C1
Огнестойкость: R120

Нормы РФ: СП 513.1325800.2022
Нормы ЕС: EN 1992-4:2018

Примечания: Подходит для влажного бетона
```

## Обязательные поля

| Поле | Описание | Пример |
|------|----------|--------|
| Производитель | Название компании | Hilti, Fischer, Mungo |
| Продукт | Название продукта | HIT-RE 500 V3 |
| Тип анкера | Тип крепления | chemical, mechanical_expansion |
| Размер | Диаметр резьбы | M8, M10, M12, M16 |

## Опциональные поля

Все остальные поля опциональны. Бот сохранит то, что есть, а недостающее можно дополнить позже.

## Типы анкеров

| Код | Описание |
|-----|----------|
| chemical | Химический (клеевой) |
| mechanical_expansion | Механический распорный |
| mechanical_undercut | Механический подрезной |
| screw | Шуруп по бетону |
| cast_in | Закладной |

## Как работает сохранение

1. Данные парсятся из свободного текста
2. Запись добавляется в `databases/anchors/anchor-database.json`
3. Создаётся Markdown файл в `thoughts/learnings/`
4. Обновляется [[MOC/MOC-anchors|MOC: Анкеры]]
5. Ссылки связываются автоматически
