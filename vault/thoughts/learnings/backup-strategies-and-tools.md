# Стратегии бэкапа и лучшие инструменты для восстановления данных

## Правило 3-2-1 (и его эволюция)

Золотой стандарт бэкапа:
- **3** копии данных (оригинал + 2 копии)
- **2** разных типа носителей (SSD + облако, HDD + NAS)
- **1** копия off-site (удалённо, в другом месте)

### Расширение 3-2-1-1-0
- Дополнительная **1** — immutable (неизменяемая) копия (WORM / air-gapped)
- **0** ошибок при проверке восстановления

## Лучшие инструменты

### Для полного образа диска (bare-metal recovery)
- **Clonezilla** — бесплатный, загружается с USB, клонирует диск целиком (включая скрытые разделы), бэкапит только занятые блоки
- **Acronis True Image** — коммерческий лидер, full disk image + cloud + cybersecurity
- **Macrium Reflect** — популярен на Windows, быстрое создание образов

### Инкрементальные бэкапы с дедупликацией
- **BorgBackup** — дедупликация, сжатие, AES-256 шифрование (Linux/macOS/BSD)
- **Restic** — быстрый, шифрованный, поддержка облачных хранилищ (S3, B2, GCS)
- **Duplicati** — GUI, шифрование, облачные бэкенды, кроссплатформенный

### Системные снапшоты
- **Timeshift** (Linux) — аналог Time Machine / System Restore, инкрементальные снапшоты через rsync
- **Time Machine** (macOS) — встроенное решение Apple
- **Windows Backup / File History** — встроенное решение Windows

### Enterprise / NAS
- **Veeam** — лидер enterprise бэкапов (VM, cloud, SaaS)
- **Synology Active Backup** — если есть Synology NAS

## Сценарий полного восстановления компьютера

1. **Подготовка** — создать загрузочную USB-флешку с Clonezilla или Acronis
2. **Регулярный бэкап** — ежедневные инкрементальные бэкапы через BorgBackup/Restic на внешний диск + облако
3. **Еженедельный full image** — полный образ диска через Clonezilla
4. **При потере данных**:
   - Загрузиться с USB → восстановить образ диска (bare-metal recovery)
   - Или установить чистую ОС → восстановить файлы из Borg/Restic
5. **Проверка** — раз в месяц тестировать восстановление из бэкапа

## Рекомендуемая комбинация для домашнего пользователя

| Задача | Инструмент |
|--------|-----------|
| Full disk image | Clonezilla (бесплатно) или Acronis |
| Ежедневные файлы | BorgBackup или Restic → внешний HDD |
| Облачная копия | Restic → Backblaze B2 (~$5/мес за 1TB) |
| Системные снапшоты | Timeshift (Linux) / Time Machine (macOS) |
