# Сценарии применения Claude AI для инженера-конструктора в девелопменте

## 1. Проверка проектной документации
- Вычитка пояснительных записок на логику, полноту, противоречия
- Сверка разделов ПД между собой (АР vs КР vs ИОС)
- Чек-листы по СП, ГОСТ, нормативам
- Проверка спецификаций на дублирование и пропуски

## 2. Составление документов
- Генерация шаблонов ТЗ, писем подрядчикам, служебных записок
- Пояснительные записки к разделам проекта
- Ответы на замечания экспертизы — черновики
- Протоколы совещаний по фото/аудио заметкам

## 3. Оценка трудозатрат и сроков
- Декомпозиция проекта на этапы с оценкой часов
- Сравнение план/факт по аналогичным проектам
- Расчёт загрузки команды

## 4. Финансовая оценка и бизнес-модели
- Предварительная оценка стоимости строительства по укрупнённым показателям
- Анализ ТЭП проекта
- Сравнение вариантов конструктивных решений по стоимости
- ROI-анализ технических решений

## 5. Работа с нормативной базой
- Быстрый поиск по СП, СНиП, ГОСТ
- Разъяснение требований нормативов
- Проверка актуальности ссылок на нормативы в проекте

## 6. Проверка расчётов
- Верификация логики расчётов (не замена ПК, а second opinion)
- Проверка исходных данных и допущений
- Анализ результатов расчёта на адекватность

## 7. Коммуникации
- Составление писем заказчику/подрядчику
- Формулировка замечаний к смежным разделам
- Подготовка презентаций для руководства

## 8. Обучение и развитие
- Объяснение незнакомых методик расчёта
- Разбор сложных пунктов нормативов
- Подготовка к аттестации НОСТРОЙ
