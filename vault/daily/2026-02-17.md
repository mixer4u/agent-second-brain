## Алгоритм оплаты через Bybit Card

Аккаунт Bybit привязан к российскому номеру. Карта Bybit официально недоступна для РФ, но есть обходной путь через регион Казахстан.

### Пошаговый алгоритм

#### Этап 1: Подготовка аккаунта Bybit
1. Зайти в аккаунт Bybit (работает для россиян, торговля доступна)
2. Пройти KYC Level 1 если ещё не пройден (паспорт РФ, автоматическая проверка ~30 мин)

#### Этап 2: Смена региона для карты
1. Зайти в раздел Bybit Card
2. При заполнении анкеты указать регион — Казахстан
3. Для подтверждения номера использовать SMS-активатор (sms-activate.org или аналог):
   — Пополнить баланс SMS-активатора (~50-100₽)
   — Купить временный казахстанский номер
   — Получить SMS с кодом подтверждения
   — Ввести код на Bybit
4. Выбрать виртуальную карту (активируется мгновенно)

#### Этап 3: Пополнение карты
1. Купить USDT через P2P на Bybit:
   — Раздел «Купить крипто» → P2P-торговля
   — Выбрать «Покупка USDT» → оплата через банковский перевод/СБП
   — Фильтр «Только проверенные мерчанты»
   — Перевести рубли, получить USDT на Funding Account
2. Перевести USDT на баланс карты (Funding → Card)
3. При оплате USDT автоматически конвертируется в валюту платежа

#### Комиссии
- Выпуск карты: 0 USDT (промо 2026)
- Обслуживание: 0 при обороте от $100/мес, иначе 5 USDT/мес
- Конвертация крипто → фиат: 0.9%
- Обмен валют: 2%
- Снятие наличных: 2% (свыше $100/мес)

#### Лимиты
- До $5,000 в день
- До $25,000 в месяц

#### Что работает с картой Bybit
- Spotify, Netflix, Booking
- Google Ads, ChatGPT (зависит от BIN)
- Зарубежные онлайн-магазины

#### Что НЕ работает
- Apple Pay / Google Pay — не привязывается
- App Store, Google Play — могут отклонять

#### Риски
1. Аккаунт могут заблокировать при несовпадении региона KYC и карты
2. SMS-активатор — серая зона, номер может быть использован повторно
3. P2P-покупка USDT — риск фрода контрагента (использовать только проверенных)

## Безопасность авторизации Яндекса через Obsidian-плагин

### Контекст
Плагин Remotely Save позволяет синхронизировать vault Obsidian с Яндекс Диском через OAuth 2.0 или WebDAV.

### Как это работает
- OAuth: плагин запрашивает доступ к Яндекс Диску, Яндекс выдаёт токен (ключ доступа)
- WebDAV: создаётся отдельный пароль приложения (не основной пароль Яндекса)
- Токен/пароль хранится в файле data.json внутри папки плагина

### Риски
1. Токен хранится в открытом виде в data.json — доступен любому, кто получит доступ к файлу
2. Плагин open-source, но не проходил формальный аудит безопасности
3. OAuth-токен даёт доступ ко всему Яндекс Диску (не только к папке vault)
4. Если токен утечёт — злоумышленник получит доступ к файлам на диске
5. Плагин обновляется автоматически — теоретически возможна атака через обновление

### Меры безопасности
1. Использовать WebDAV + пароль приложения вместо OAuth (ограниченный доступ)
2. Создать отдельный аккаунт Яндекса только для синхронизации
3. Не хранить на этом диске конфиденциальные файлы
4. Включить E2E-шифрование в настройках Remotely Save
5. Не коммитить data.json в git
6. Периодически отзывать и обновлять токены на id.yandex.ru/security

### Вывод
Умеренно безопасно при соблюдении мер. WebDAV + пароль приложения безопаснее OAuth. Идеально — выделенный аккаунт Яндекса + E2E-шифрование.

## Скрытая анкерная точка через вентфасад с аграфами (вентзазор 300 мм)

### Проблема
Нужно организовать анкерную точку для промальпа/страховки через вентилируемый фасад с аграфами. Вентзазор 300 мм. Крюк должен быть полностью скрыт за облицовкой.

### Решение: Комбинированная система «анкер в стену + удлинитель + съёмная панель»

#### Вариант 1: Забивной анкер с внутренней резьбой + шпилька-удлинитель

1. В несущую стену устанавливается **забивной анкер с внутренней резьбой** (заподлицо с поверхностью стены)
2. Через вентзазор 300 мм проходит **резьбовая шпилька M16 или M20** с проушиной (рым-болт) на конце
3. Шпилька вкручивается в анкер только на время работ
4. Облицовочная панель в этом месте делается **съёмной** (на быстросъёмных кляммерах или магнитных фиксаторах вместо стандартных аграфов)
5. Перед работами панель снимается → вкручивается шпилька с проушиной → после работ шпилька снимается → панель возвращается на место

**Плюсы:** полностью невидимо, многократное использование, стандартный крепёж
**Минусы:** нужен доступ к панели для снятия, требуется расчёт на вырыв

#### Вариант 2: Стационарный скрытый кронштейн (по типу Innotech)

1. К несущей стене крепится **усиленный кронштейн** (аналог фасадного, но рассчитанный на нагрузку 15-25 кН)
2. Кронштейн проходит через вентзазор и заканчивается **анкерной проушиной** на уровне подконструкции
3. Проушина располагается **за плоскостью облицовки** в вентзазоре
4. В облицовочной панели напротив проушины — **съёмная заглушка** или съёмная панель на аграфах
5. Доступ через снятие панели/заглушки

**Плюсы:** стационарная конструкция, не нужно каждый раз вкручивать шпильку
**Минусы:** дороже, нужен индивидуальный проект

#### Вариант 3: Система Innotech TAURUS / AIO

Промышленное решение от Innotech:
- **TAURUS** — рельсовая система с тонким профилем, может монтироваться скрыто за фасадом
- **AIO Lifeline** — линейная система с подвижными точками крепления
- Обе системы специально разработаны для фасадов и предусматривают скрытый монтаж
- STA-стойки обеспечивают соединение с подконструкцией через вентзазор любой глубины

### Критические требования

1. **Расчёт на вырыв** — анкер в несущей стене должен держать минимум 15 кН (ГОСТ Р ЕН 795)
2. **Материал стены** — бетон, кирпич (для газобетона нужен химический анкер)
3. **Испытания** — обязательны статические испытания каждой точки на вырыв
4. **Шпилька** — нержавеющая сталь A4 (AISI 316), длина ~350-400 мм с учётом утеплителя
5. **Съёмная панель** — не должна нарушать вентиляцию и гидрозащиту

### Рекомендуемые поставщики

- **Innotech** (Австрия) — системы TAURUS, AIO для скрытого монтажа на фасадах
- **Petzl** — анкерные устройства для промальпа
- **Safe-Tec / AltusPro** — стационарные анкерные точки, проектирование
- **АльпИндустрия-ПРО** — консультации по установке

### Вывод
Оптимальное решение: забивной анкер с внутренней резьбой в несущую стену + съёмная облицовочная панель + вкручиваемая шпилька с проушиной на время работ. Для постоянных работ — система Innotech TAURUS за облицовкой.
